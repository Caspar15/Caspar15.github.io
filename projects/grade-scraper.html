學業成績曲線圖

此項目使用 Python 的 Matplotlib 與 Seaborn 庫，繪製了一個展示 GPA 與系排名變化趨勢的曲線圖。圖表具有漸變陰影效果，能夠直觀呈現您的學習成果。

功能介紹

  • 雙 Y 軸曲線圖： 同時展示 GPA 與系排名的變化趨勢。
  • 數據標註： 在數據點上標註具體的 GPA 值與排名資訊。
  • 漸變陰影效果： 為曲線下方添加漸變色陰影，增強視覺效果。
  • 自適應標註： 智能調整標註位置，避免重疊。
  • 美觀主題與配色： 採用 Seaborn 主題風格與協調配色方案。

環境要求

  • Python 3.x
  • Matplotlib
  • Seaborn
  • NumPy

安裝指南

  1. 克隆專案：

    git clone https://github.com/yourusername/academic-performance-plot.git

  2. 安裝所需庫：

    pip install matplotlib seaborn numpy

  3. 確認系統已安裝 Microsoft JhengHei 字體（或使用其他中文字體）。

使用方法

  1. 運行代碼：

    python your_script_name.py

  2. 運行後將彈出窗口顯示學業成績曲線圖。
