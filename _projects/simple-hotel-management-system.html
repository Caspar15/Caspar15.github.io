Introduction

The Hotel Management System is a Java-based application designed to manage hotel bookings, room availability, and customer details. This system allows hotel staff to efficiently handle reservations, check-ins, and check-outs, providing a seamless experience for both staff and guests.

Features

  • Room Booking: Book luxury and deluxe rooms (both single and double).
  • Room Availability: Check the availability of different types of rooms.
  • Customer Details: Record and manage customer details for single and double room bookings.
  • Billing: Generate and print bills for customers at check-out.
  • Data Persistence: Save and load data to/from a file for persistence.

Installation

Prerequisites

  • Java Development Kit (JDK) 8 or higher
  • Git

Steps

  1. Clone the repository:

    git clone https://github.com/Caspar15/Simple-hotel-management-system.git

  2. Navigate to the project directory:

    cd hotel-management-system

  3. Compile the project:

    javac -d bin src/finalproject/*.java

  4. Run the project:

    java -cp bin finalproject.Main

Usage

Booking a Room

  1. Launch the application.
  2. Choose the option to book a room.
  3. Select the type of room you want to book (Luxury/Deluxe, Single/Double).
  4. Enter customer details as prompted.
  5. Confirm the booking.

Checking Room Availability

  1. Launch the application.
  2. Choose the option to check room availability.
  3. Select the type of room to check availability.
  4. View the number of available rooms.

Check-Out

  1. Launch the application.
  2. Choose the option to check-out.
  3. Enter the room number to check-out.
  4. Confirm check-out and view the bill.
